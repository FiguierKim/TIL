### 들여쓰기

파이썬의 들여쓰기는 매우 중요 -> 들여쓰기로 코드 블록을 나타내기 때문에

```python
if 3 > 2:
  print("Three")
```

이건 실행 되지만

```python
if 3 > 2:
print("Three") 
```

이건 실행이 안됨
