



```java
public class HelloWorld {
  public static void main(String[] args) {
    system.out.println("Hello, World!");
  }
}
```



`public` 

접근제어자

`class` 

객체 지향 프로그래밍의 기본 단위

`HelloWorld` 

클래스 이름

`static`

바로 실행 가능하게 만들어 줌

`void`

리턴 값의 유형, 돌려줄 값이 없다는 뜻

`main`

정의할 메소드의 이름

자바 프로그램 실행시 첫 번째로 `main` 을 찾아 실행시킴

모든 프로그램에 포함

`String[] args`

파라미터

args라는 String 배열

`System`

자바에서 미리 만들어둔 클래스

`out`

출력 기능

`println()`

출력 메소드

괄호 안 입력값을 출력한 뒤 줄 바꿈
