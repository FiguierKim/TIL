# Day 13

## API

라이브러리(library)라고 불리기도 한다. 

자주 사용되는 클래스 및 인터페이스의 모음

String, System 클래스 또한 API의 클래스이다



## java.lang

Object - 자바 클래스 최상위

System - 표준 입력 장치로 데이터를 입력받을 때 사용

Class - 클래스를 메모리로 로딩할 때

String - 문자열 저장

StringBuffer, StringBuilder - 문자열 저장, 문자열 조작

Math - 수학 함수



## java.util

Arrays - 배열 비교, 복사, 정렬, 찾기

Calendar - 날짜와 시간 획득

Date - 날짜와 시간 저장

StringTokenizer - 특정 문자로 구분된 문자열 추출

Random - 난수 획득



### equals()

객체 비교

true / false 값 리턴

`==` 과 동일



### hashCode()

객체를 식별할 하나의 정수값



### toString()

객체의 문자 정보를 리턴



### clone()

원본 객체의 필드값과 동일한 값을 가지는 새로운 객체 생성
